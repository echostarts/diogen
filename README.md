# ДИОГЕН

Браузерный bullet-heaven (жанр Vampire Survivors): Афины, ~350 г. до н. э.
Вы — Диоген Синопский. Катаетесь по агоре в бочке и пять минут отбиваетесь от
волн возмущённых граждан, пока не явится Александр — лично. Вся графика
процедурная (Canvas-примитивы под чернофигурную вазопись), звук — синтез
WebAudio плюс CC0-сэмплы из `public/audio/`: музыка на греческих инструментах
(ран/босс/финал), джинглы эволюций, деревянный тук тарана, фоли ударов,
клики меню и гул возмущённой толпы. У каждого звука остался синтез-фолбэк,
если файлы не догрузились. Остальная отобранная библиотека внешних ассетов
(частицы, иконки, тайлсеты-референсы) — в `assets/`, реестр и лицензии —
в `assets/ASSETS.md`.

## Запуск

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # прод-сборка в dist/
npm test           # Playwright-набор: загрузка, инпут, fps, бот-раны, босс
```

Полезные query-флаги (для отладки и тестов): `?bot=1` — автопилот,
`?seed=N` — детерминированный ран, `?speed=N` — ускорение симуляции,
`?stress=1` — перф-сцена (300 врагов), `?bosshp=N` — подмена HP босса.
Сиды в `tests/game.spec.ts` пришпилены к текущему балансу — после правок
логики переподберите их скриптом `node scripts/calibrate.mjs "1 2 3 4 5 6 7 8" 8`.

## Управление

| Клавиши | Действие |
| --- | --- |
| WASD / стрелки / геймпад | катить бочку |
| ПРОБЕЛ | рывок бочкой (когда открыта «Бочка-таран» и заряд полон) |
| 1/2/3, ←→ + ENTER | выбор апгрейда на левел-апе |
| ESC / P | пауза |
| M (или клик по иконке) | звук вкл/выкл |
| L | лавка Диогена (на тайтле) |
| R / ENTER | рестарт с экрана финала |

На тач-устройствах: палец на левой половине экрана — виртуальный стик,
тап по правой половине — рывок, карты апгрейдов и кнопки берутся тапом.

Атаки автоматические. Оружия: Фонарь («ищу человека», круг света тикает
уроном), Плевок цинизма, Псы, Бочка-таран. У каждого оружия есть эволюция
(оружие V + родственная пассивка III): СОЛНЦЕ, ДИАТРИБА, СВОРА КИНИКОВ,
ПИФОС. Враги: торговец (быстрый, x2 XP), стражник, платоник-танк, софист
с медленными свитками; с 1:40 попадаются элитные граждане (кольцо охры,
втрое толще, шесть оливок). На 2:30 приходит мини-босс — сам Платон. Пока
Александр стоит в круге света Фонаря, он получает +50% урона.

За раны начисляются черепки (1 за 10 переубеждённых, +20 за победу) —
в «Лавке Диогена» на тайтле они обмениваются на постоянные бонусы.

Героев двое (переключение ◀ ▶ на тайтле): **Диоген** — бочка, фонарь и
таран; **Гиппархия** (открывается за 150 черепков) — пешком и без тарана,
зато быстрее, стартует с Плевком, имеет врождённое +1 пробитие, а на
ПРОБЕЛ делает «шаг в сторону» с краткой неуязвимостью.

## Структура кода

Ядро — `src/engine/`: маленькие самодостаточные модули без знания об игре.
`input.ts` собирает клавиатуру и геймпад в вектор движения и фронты действий,
`audio.ts` синтезирует эффекты из осцилляторов и шума (ленивая инициализация
по первому жесту) и крутит файловую музыку из `public/audio/` — луп рана,
кроссфейд на босса, джинглы финала и эволюций, а при недоступности файлов
откатывается на встроенный степ-секвенсор, `rng.ts` — детерминированный mulberry32 для воспроизводимых
ранов, `shash.ts` — spatial hash на typed arrays (counting sort), который
перестраивается каждый тик без единой аллокации.

Игровое состояние живёт в `src/game/world.ts`: один класс World держит все
пулы (враги, снаряды, оливки, частицы, цифры урона) с swap-remove-уплотнением
и хелперы урона/опыта/эффектов; `reset()` переинициализирует поля без
пересоздания объектов — отсюда рестарт без перезагрузки страницы. Системы —
чистые функции над World: `player.ts`, `enemies.ts` (спавнер волн, ИИ,
расталкивание), `weapons.ts` (все четыре оружия), `boss.ts` (фазы Александра),
`pickups.ts`, `fx.ts`, `upgrades.ts` (17 карт с весами выпадения, включая
четыре эволюции) и `bot.ts` — автопилот для плейтестов. `game.ts` — машина
состояний (тайтл → ран → левел-ап/пауза → финал, плюс лавка), мета-прогресс
в localStorage и порядок вызова систем.

Рендер отделён от логики: `render.ts` рисует мир по World в device-координатах
(камера, тряска, спрайты без save/restore в горячих циклах), `sprites.ts` один
раз при старте пре-рендерит силуэты врагов и фоновый тайл в offscreen-канвасы,
`hud.ts` и `screens.ts` — интерфейс с кэшем строк (ноль аллокаций в кадре).
Точка входа `main.ts` крутит цикл: логика фиксированным шагом 60 Гц с
аккумулятором, рендер по requestAnimationFrame, letterbox под любое окно и
диагностика `window.__diag` для Playwright.

## Деплой

GitHub Actions (`.github/workflows/deploy.yml`) на каждый push в `main`
собирает Vite (`base: './'` — пути относительные) и публикует на GitHub Pages.
В настройках репозитория должно быть включено: Settings → Pages → Build and
deployment → Source: **GitHub Actions**.
